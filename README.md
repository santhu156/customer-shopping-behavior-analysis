# Customer Shopping Behavior Analysis

## Overview

This project explores customer shopping behavior using transactional data from **3,900 purchases** across multiple product categories. The objective was to transform raw data into meaningful business insights through data cleaning, SQL analysis, and interactive visualization.

The project demonstrates an end-to-end data analytics workflow using **Python**, **SQL Server Express**, and **Power BI**, covering everything from data preprocessing to business reporting.

---

## Project Objectives

* Clean and prepare raw customer transaction data
* Perform exploratory data analysis (EDA)
* Store the cleaned dataset in SQL Server Express
* Answer business questions using SQL
* Build an interactive Power BI dashboard
* Generate actionable business recommendations

# Project Workflow

## 1. Data Cleaning & Preparation (Python)

The first stage focused on preparing the dataset for analysis.

Tasks performed:

* Imported and explored the dataset using Pandas
* Checked data types and summary statistics
* Identified missing values
* Imputed missing review ratings using the median rating within each product category
* Renamed columns using snake_case naming conventions
* Created new features such as **Age Group** and **Purchase Frequency**
* Removed redundant columns to improve data consistency
* Loaded the cleaned dataset into SQL Server Express for further analysis

---

## 2. Business Analysis (SQL Server Express)

Using SQL, several business questions were answered to better understand customer behavior.

### Analysis Performed

* Revenue generated by customer gender
* High-spending customers who used discounts
* Highest-rated products
* Average spending by shipping method
* Spending comparison between subscribers and non-subscribers
* Products with the highest discount usage
* Customer segmentation based on purchase history
* Top-selling products within each category
* Relationship between repeat purchases and subscriptions
* Revenue contribution by age group

# Key Insights

* Male customers generated higher overall revenue than female customers.
* Express shipping customers had a slightly higher average purchase amount.
* Gloves, Sandals, Boots, Hat, and Skirt received the highest average customer ratings.
* Loyal customers represented the largest customer segment.
* Clothing contributed the highest overall revenue.
* Young adults generated the greatest share of total sales.
* Several products showed a high dependence on discounts, highlighting opportunities to optimize promotional strategies.

---

# Business Recommendations

Based on the analysis, the following recommendations were identified:

* Increase subscription adoption through exclusive membership benefits.
* Reward repeat customers with loyalty programs.
* Optimize discount strategies to improve profitability.
* Promote top-rated products in marketing campaigns.
* Focus marketing efforts on customer segments that contribute the highest revenue.
